re-enable-copy
==============

Disables copy/paste disabling scripts in certain websites.

Namely:
- http://folha.uol.com.br

Download it from: https://chrome.google.com/webstore/detail/re-enable-copy/kpojaicmdgpmoemkdlbnmokolafjmcbn
